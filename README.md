## üöÄ Jinsi ya Kuanza

### Mahitaji
- Node.js 18+ 
- npm au yarn
- Kivinjari cha kisasa

### Usakinishaji

1. **Clone Repository**
\`\`\`bash
git clone https://github.com/ezradaniel/tanzania-radio-hub.git
cd tanzania-radio-hub
\`\`\`

2. **Sakinisha Dependencies**
\`\`\`bash
npm install
# au
yarn install
\`\`\`

3. **Anzisha Development Server**
\`\`\`bash
npm run dev
# au
yarn dev
\`\`\`

4. **Fungua Kivinjari**
\`\`\`
http://localhost:3000
\`\`\`
